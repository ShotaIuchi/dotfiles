# Ghostty 設定

## 基本設定

| 設定 | 値 |
|---|---|
| テーマ | TokyoNight Night |
| フォント | UDEV Gothic NFLG |
| フォントサイズ | 13 |
| カーソル | bar（点滅なし） |
| パディング | 4px（上下左右） |

## キーバインド

| 設定 | 説明 |
|---|---|
| `macos-option-as-alt = true` | Option キーを Alt として使用（fzf の `Alt+C` 等に必要） |
| `ctrl+m = text:\n` | Ctrl+m を改行に（Emacs の C-m 互換） |
