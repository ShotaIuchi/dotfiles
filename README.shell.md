# シェル設定

bash/zsh 共通設定（`.shell_common`）。インタラクティブシェル起動時に tmux を自動開始。

## エイリアス

### ファイル操作 (eza)

| エイリアス | コマンド | 説明 |
|---|---|---|
| `ls` | `eza` | - |
| `ll` | `eza -la --git --icons` | 詳細表示 + Git ステータス |
| `la` | `eza -a` | 隠しファイル含む |
| `l` | `eza -1` | 1行1ファイル |
| `lt` | `eza --tree --level=2` | ツリー表示（2階層） |

### エディタ

| エイリアス | コマンド |
|---|---|
| `vi` / `vim` | `nvim` |

### ファイル表示 (bat)

| エイリアス | コマンド | 説明 |
|---|---|---|
| `cat` | `bat -pp` | シンタックスハイライト付き表示 |
| `cat -n` 等 | `bat -n` | オプション付きはそのまま bat に渡す |
| `md` | `glow -p` | Markdown ビューア |

### 安全装置

| エイリアス | コマンド |
|---|---|
| `rm` | `rm -i` |
| `cp` | `cp -i` |
| `mv` | `mv -i` |

## fzf

### デフォルトオプション

- 画面下部 40%、ボーダー付き、逆順表示
- `fd` がある場合は `find` の代わりに使用（高速、`.git` 除外）

### fzf キーバインド

| キー | 機能 |
|---|---|
| `Ctrl+R` | コマンド履歴検索 |
| `Ctrl+T` | ファイル検索（bat プレビュー付き） |
| `Alt+C` | ディレクトリ選択して cd |

### fzf 内の操作キー

| キー | 機能 |
|---|---|
| `C-n` / `C-p` | 上下移動 |
| `Alt-j` / `Alt-k` | 上下移動 |
| `Alt-n` / `Alt-p` | ページ送り |
| `C-v` / `Alt-v` | プレビュー上下スクロール |
| `C-k` | 行削除 |

### fzf 関数

| 関数 | 説明 |
|---|---|
| `fbr` | fzf でブランチ選択して checkout |
| `flog` | fzf で Git ログ表示（diff プレビュー付き） |
| `fcat` | fzf でファイル選択して bat 表示 |
