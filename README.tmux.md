# tmux / Zellij 設定

tmux を優先使用、未インストール時は Zellij にフォールバック。
両方とも `C-x` プレフィックスで Emacs 互換のキーバインドに統一。

## tmux キーバインド

プレフィックス: `C-x`

### ペイン操作

| キー | 機能 |
|---|---|
| `C-x 2` | 水平分割（下） |
| `C-x 3` | 垂直分割（右） |
| `C-x 0` | 現在のペインを閉じる |
| `C-x 1` | ズームトグル |
| `C-x 4` | ペインサイズを均等化 |

### ペイン移動

| キー | 機能 |
|---|---|
| `C-x o` | 次のペインへ |
| `C-x n` | 前のペインへ |
| `C-x h/j/k/l` | 方向指定で移動 |
| `C-x ←↑↓→` | 方向キーで移動 |

### フローティング

| キー | 機能 |
|---|---|
| `C-x 6` | フローティングシェル（中央 80%） |

### セッション

| キー | 機能 |
|---|---|
| `C-x d` | デタッチ |
| `C-x C-x` | アプリに C-x を送信 |
| `C-x C-g` | キャンセル |

### コピーモード

| キー | 機能 |
|---|---|
| `M-n` | ページダウン |
| `M-p` | ページアップ |
| マウスドラッグ | 選択してコピー |

## tmux 設定

| 設定 | 値 |
|---|---|
| テーマ | Tokyo Night |
| ステータスバー | 非表示 |
| マウス | 有効 |
| スクロールバック | 50000行 |
| ペイン開始番号 | 1 |
| true color | ghostty 対応 |

## Zellij キーバインド

tmux と同じ `C-x` プレフィックスで同じ操作体系。

| キー | 機能 |
|---|---|
| `C-x 2` | 水平分割（下） |
| `C-x 3` | 垂直分割（右） |
| `C-x 0` | ペインを閉じる |
| `C-x 1` | フルスクリーントグル |
| `C-x o` | 次のペインへ |
| `C-x n` | 前のペインへ |
| `C-x h/j/k/l` | 方向指定で移動 |
| `C-x 4` | フローティングペイン |
| `C-x 6` | フローティングシェル |
| `C-x d` | デタッチ |
| `C-x C-x` | アプリに C-x を送信 |
| `C-x C-g` / `Esc` | キャンセル |
