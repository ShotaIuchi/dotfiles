# Neovim キーバインド

プラグインマネージャーは lazy.nvim。テーマは tokyonight (night)。

## プラグイン

| プラグイン | 用途 |
|---|---|
| tokyonight.nvim | カラースキーム |
| lualine.nvim | ステータスライン |
| gitsigns.nvim | Git変更表示（sign column） |
| nvim-treesitter | シンタックスハイライト |
| vim-fugitive | Gitコマンド |
| Comment.nvim | コメントトグル |
| nvim-autopairs | 括弧自動閉じ |

## リーダーキー

`Space` がリーダーキー。

## ノーマルモード

### ファイル操作

| キー | 機能 |
|---|---|
| `<leader>w` | ファイル保存 |
| `<leader>q` | 終了 |
| `<leader>Q` | 全終了（強制） |

### 検索

| キー | 機能 |
|---|---|
| `Esc` | 検索ハイライトをクリア |

### ウィンドウ移動

| キー | 機能 |
|---|---|
| `C-h` | 左のウィンドウへ |
| `C-j` | 下のウィンドウへ |
| `C-k` | 上のウィンドウへ |
| `C-l` | 右のウィンドウへ |

### ウィンドウリサイズ

| キー | 機能 |
|---|---|
| `C-Up` | 高さ +2 |
| `C-Down` | 高さ -2 |
| `C-Left` | 幅 -2 |
| `C-Right` | 幅 +2 |

### バッファ移動

| キー | 機能 |
|---|---|
| `S-h` | 前のバッファ |
| `S-l` | 次のバッファ |
| `<leader>bd` | バッファ削除 |

### 診断

| キー | 機能 |
|---|---|
| `[d` | 前の診断へ |
| `]d` | 次の診断へ |
| `<leader>d` | 診断をフロート表示 |

### Git (vim-fugitive)

| キー | 機能 |
|---|---|
| `<leader>gs` | Git status |
| `<leader>gb` | Git blame |
| `<leader>gd` | Git diff（分割表示） |

### コメント (Comment.nvim)

| キー | 機能 |
|---|---|
| `gcc` | 行コメントトグル |
| `gbc` | ブロックコメントトグル |
| `gc` (Visual) | 選択範囲をコメントトグル |

## インサートモード

| キー | 機能 |
|---|---|
| `jk` | Escape |
| `jj` | Escape |

## ビジュアルモード

| キー | 機能 |
|---|---|
| `J` | 選択行を下に移動 |
| `K` | 選択行を上に移動 |
| `<` / `>` | インデント（選択維持） |
| `p` | ペースト（レジスタ上書きなし） |

## オプション

| 設定 | 値 |
|---|---|
| 行番号 | relative |
| タブ幅 | 4（lua/yaml/json/js/tsは2） |
| クリップボード | システム共有 (`unnamedplus`) |
| swap/backup | 無効 |
| undofile | 有効 |
| スクロールマージン | 8行 |
